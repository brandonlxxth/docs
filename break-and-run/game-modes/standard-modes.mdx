---
title: Standard game modes
description: Race to, Sets of, Best of, and Free play modes
---

Break & Run supports four standard game modes for traditional two-player pool matches. Each mode offers different win conditions and scoring structures.

## Race to

First player to reach the target score wins the game.

### Setup
- Enter player names
- Set target score (e.g., 5, 7, 9)
- Choose who breaks first

### Gameplay
- Players alternate turns
- Score increases by 1 for each frame won
- First to reach target score wins
- Game ends immediately when target is reached

### Use cases
- Quick matches
- Tournament play
- Skill-based competition

## Sets of

Play multiple sets, each to a target score. Track set wins separately from frame scores.

### Setup
- Enter player names
- Set target score per set (e.g., 3, 5)
- Set number of sets to play
- Choose who breaks first

### Gameplay
- Each set plays like a "Race to" game
- Winner of each set is recorded
- Scores reset between sets
- All sets are played regardless of outcome

### Scoring
- Frame scores track within each set
- Set wins are tallied separately
- View frame-by-frame history for each set
- Final result shows total sets won by each player

### Use cases
- Longer matches
- Best-of-series tournaments
- Tracking performance across multiple games

## Best of

First player to win the majority of sets wins the match.

### Setup
- Enter player names
- Set target score per set
- Set total number of sets (must be odd: 3, 5, 7, etc.)
- Choose who breaks first

### Gameplay
- Similar to "Sets of" mode
- Match ends when one player wins majority of sets
- Remaining sets are not played

### Example
In a best of 5:
- First player to win 3 sets wins the match
- Match could end 3-0, 3-1, or 3-2
- Maximum 5 sets, minimum 3 sets

### Use cases
- Competitive matches
- Tournament finals
- Determining overall winner efficiently

## Free play

Practice mode with no win conditions or target scores.

### Setup
- Enter player names
- Choose who breaks first
- No target score required

### Gameplay
- Track scores indefinitely
- No winner determination
- End game manually when finished
- Full frame history maintained

### Features
- Dish tracking (break dish, reverse dish)
- Score adjustments (+1, -1)
- Frame-by-frame history
- Statistics tracking

### Use cases
- Practice sessions
- Casual play
- Skill development
- Testing new techniques

## Common features

All standard modes support:

### Dish tracking
- **Break dish**: Player dishes on the break
- **Reverse dish**: Player dishes after opponent's dish
- **Miss**: Track missed shots
- Dishes are recorded in frame history

### Ball color selection
- Red or yellow ball assignment
- Foul break handling
- Color tracking throughout game

### Score adjustments
- +1: Award frame to player
- -1: Deduct frame from player
- Manual corrections for disputes

### Game history
- Frame-by-frame breakdown
- Timestamp for each frame
- Score progression tracking
- Dish type recording

### Resume capability
- Games auto-save on each frame
- Resume from home screen if interrupted
- Preserves all game state and history
