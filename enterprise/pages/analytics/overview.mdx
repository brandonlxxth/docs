## Overview

The **Analytics** section provides real-time dashboards, reporting tools, and activity tracking to help you monitor and analyse your business operations. Access it via **Analytics** in the main navigation menu.

## Required permissions

Each page within Analytics has its own permission. See individual page documentation for specific permission requirements. Permissions are configured in **Admin > User Roles** under the **Analytics** tab.

## Activity

The Activity subsection provides tools to track and review system activity: transaction receipts, Z reports, order/product journals, checklist completions, product refusals, and preparation point operations.

<Card title="Activity" icon="chart-network" href="/enterprise/pages/analytics/activity">
  Full documentation for Receipt Explorer, Z Report Explorer, Journal, Checklist, Product Refusal, CCU Journal, and Click and Collect Journal.
</Card>

| Page | Description |
|------|-------------|
| **Receipt Explorer** | Search and view transaction receipts from terminals. |
| **Z Report Explorer** | View Z reports for end-of-day reconciliation. |
| **Journal** | Download order or product journal data as CSV. |
| **Checklist** | View checklist completion reports by operator, terminal, and time. |
| **Product Refusal** | Track product refusal events for age-restricted items. |
| **CCU Journal** | Download CCU (Customer Communication Unit) journal data. |
| **Click and Collect Journal** | Download click and collect order data. |

## Events

Manage revenue events for tracking special occasions, promotions, or time periods. Events allow you to segment and analyse sales data in reports.

<Card title="Events" icon="calendar-days" href="/enterprise/pages/analytics/events">
  Full documentation for Events, Event Categories, and Event Tags.
</Card>

| Page | Description |
|------|-------------|
| **Events** | Create and manage revenue events with date ranges, categories, tags, and outlet assignments. |
| **Event Categories** | Organise events into categories with forecast and budget values per outlet. |
| **Event Tags** | Add tags to events for additional filtering and organisation. |

## Heatmap

Visual heatmap displays for analysing sales patterns across your locations. Upload floor plans or site maps, place outlet markers, and view revenue or order data overlaid as a heatmap.

<Card title="Heatmaps" icon="map-pin" href="/enterprise/pages/analytics/heatmaps">
  Full documentation for heatmap configuration, designer, and viewing modes.
</Card>

| Page | Description |
|------|-------------|
| **Heatmaps** | View heatmap images with sales data overlaid showing activity patterns. |
| **Configuration** | Create heatmaps by uploading images and positioning outlet markers. |
