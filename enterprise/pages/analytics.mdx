## Overview

The Analytics section provides dashboards, reporting tools, and activity tracking to help you monitor and analyse your business operations.

## Analytics pages and features

### Dashboard

The Dashboard provides an overview of your business operations with key metrics and charts. Access real-time data about sales, performance, and operations.

### Reports

Access all available reports from a central location. Generate reports on sales, inventory, performance, and more with customizable date ranges and criteria.

### Generated Reports

View previously generated reports that have been saved for later reference.

### Activity

The Activity subsection provides tools to track and review system activity:

- **Receipt Explorer**: Search and view transaction receipts from your terminals
- **Z Report Explorer**: View Z reports from terminals for end-of-day reconciliation
- **Journal**: View system journal entries for audit and tracking purposes
- **Checklist**: View checklist activity and completion status
- **Product Refusal**: Track product refusal events when operators decline to sell items
- **CCU Journal**: View CCU (Customer Communication Unit) journal entries
- **Preparation Point Journal**: View preparation point activity and operations

### Events

Manage revenue events and event tracking:

- **Events**: Create and manage revenue events for tracking special occasions, promotions, or time periods
- **Event Categories**: Organize events into categories for better reporting
- **Event Tags**: Add tags to events for organization and reporting

### Heatmap

Visual heatmap features for analyzing business patterns:

- **Heatmaps**: View heatmap images showing activity patterns
- **Configuration**: Configure heatmap settings and data sources

### Configuration

- **Report Images**: Manage images used in reports, such as logos or custom graphics
