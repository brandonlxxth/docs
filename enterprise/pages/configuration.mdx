## Overview

The Configuration section contains all system configuration and management tools. This is where you set up products, layouts, operators, terminals, and other core system components.

## Incentives

Configure loyalty programs, vouchers, and promotional incentives:

- **Voucher**: Create and manage vouchers and promotional offers
- **Loyalty**: Configure loyalty point strategies and reward programs
- **Reward**: Set up reward programs and redemption rules
- **Game**: Manage game-based incentives and gamification features

## Campaign

Marketing and communication tools:

- **Messaging**: Create and manage messaging campaigns to communicate with customers or members
- **Notification Templates**: Create reusable templates for notifications and messages

## Management

The Management subsection contains the most comprehensive set of configuration options:

### Layouts and Menus

- **Layouts**: Configure terminal layouts for point-of-sale displays
- **Table Layout**: Set up table-based layouts for table service operations
- **Menus**: Manage sale popup menus that appear on terminals
- **Self Service Menu**: Configure self-service kiosk menus for customer ordering

### Products and Pricing

- **Products**: Manage your complete product catalog
- **Product Label**: Configure product labels and display settings
- **Product Prices**: Set and manage product pricing across locations
- **Price Bands**: Configure price band tiers for different customer segments
- **Price Band Scheduler**: Schedule price band changes for promotions or time-based pricing
- **Price Band Groups**: Organize price bands into groups
- **Product Groups**: Organize products into groups for easier management
- **Product Challenge**: Configure product restrictions and age verification requirements

### Operations

- **Tenders**: Configure payment tender types (cash, card, etc.)
- **Sessions**: Manage operational sessions and session settings
- **Operators**: Manage operator accounts and access
- **Operator Roles**: Configure operator role permissions and capabilities
- **Outlet Type**: Define outlet types to standardize configurations

### Receipts and Printing

- **Receipts**: Configure receipt templates, printing settings, and receipt formatting

### Resources and Settings

- **Resources**: Manage system resources and assets
- **Allergens**: Configure allergen information for products
- **Course**: Set up course configurations for menu organization
- **Product Tags**: Manage product tags for categorization and filtering
- **Waste Reason**: Configure waste reason codes for tracking product waste

### Pay at Table

<Warning>
**Requires Pay at Table Module**: This feature requires the Pay at Table module, which is charged separately.
</Warning>

- **QR Code Generator**: Generate QR codes for pay-at-table functionality, allowing customers to pay directly from their mobile devices

### Receipt Portal

<Warning>
**Requires Receipt Portal Module**: This feature requires the Receipt Portal module, which is charged separately.
</Warning>

- **QR Code Generator**: Generate QR codes for receipt portal access, allowing customers to view digital receipts

### Checklist

- **Configuration**: Set up checklist configurations for operational procedures
- **Dashboard**: View checklist dashboard with completion status and metrics

### Messaging

- **Send Message Templates**: Create server message templates for system notifications
- **Terminal Message Templates**: Create terminal message templates for operator communications
- **Send Message to POS**: Send messages directly to terminals in real-time
- **Message History**: View message history and delivery status

## Vision

Vision configuration for different display types and customer-facing screens:

### PoS (Point of Sale)

- **Customer Screen**: Configure customer-facing screens at point of sale
- **File Upload**: Upload assets (images, videos) for PoS displays

### Kiosk

- **Home**: Configure kiosk home screen layout and content
- **Sales**: Configure kiosk sales screen for product display
- **File Upload**: Upload assets for kiosk displays

### Quick Serve

- **Quick Serve**: Configure quick serve display screens
- **File Upload**: Upload assets for quick serve displays

### General Vision

- **Assignment**: Assign vision assets to specific locations or terminals
- **Asset**: Manage vision assets and media files
- **Vision Scheduler**: Schedule vision content changes for different times or events

## Terminal

Terminal management and monitoring:

- **Hierarchy**: View and manage terminal hierarchy and organization
- **Status**: Monitor terminal status and connectivity
- **Session Activity**: View terminal session activity and usage
- **Session Management**: Manage terminal sessions, including closing sessions remotely
- **Jobs**: View and manage terminal jobs and tasks
- **Security**: Configure terminal security settings and access controls

## Mobile App

- **Board**: Configure mobile app board settings for staff mobile applications

## Admin

Administrative and security tools:

- **Audit**: View system audit logs for security and compliance
- **Users**: Manage user accounts and access
- **User Roles**: Configure user role permissions and capabilities
- **Restriction**: Manage system restrictions and access controls
- **Connected Account**: Manage connected accounts for integrations
- **Client Devices**: Manage client device registrations and access
