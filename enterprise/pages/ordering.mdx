## Overview

The Ordering section manages online ordering, self-service kiosk ordering, and preparation point operations.

<Warning>
**Requires Ordering or Click & Collect Module**: This section requires either the Ordering module or the Click & Collect module, which are charged separately. If you don't see this section in your navigation, contact your administrator or [Kappture Sales](/contact) to enable one of these modules.
</Warning>

## Required permissions

Ordering permissions are configured in **Admin > User Roles** under the **Online Ordering** tab.

| Feature | User Role Permission | Tab in User Role Config |
|---------|---------------------|------------------------|
| Order Event | Order Event | Online Ordering |
| Configuration | Configuration | Online Ordering (Catalog Ordering) |
| Dashboard | Dashboard | Online Ordering (Catalog Ordering) |
| Order Status | Order Status | Online Ordering (Catalog Ordering) |
| Preparation Point Stock | Preparation Point Stock | Online Ordering (Catalog Ordering) |
| QR Code Generator | QR Code Generator | Online Ordering (Catalog Ordering) |
| Future Order Close | Future Order Close | Online Ordering (Catalog Ordering) |

## Ordering pages and features

### Online Event

Manage online ordering events. Create and configure events that allow customers to place orders online, set event dates and times, and manage event-specific settings.

### Dashboard

The **ordering dashboard** (Ordering > Dashboard) lets you **view** preparation point orders. You can monitor:
- Order status and queue
- Preparation point activity
- Order fulfillment metrics
- Real-time order updates

The dashboard is view-only. To **update** the status of a preparation point order (e.g. mark courses done), use a **KMS terminal** (Kitchen Management Screen); see [Terminal](/enterprise/pages/terminal/overview#kitchen-management-screen-kms).

### Product Stock

Manage preparation point stock levels. View and update stock available at preparation points for order fulfillment.

### Self Service Ordering

Configuration for self-service kiosk and mobile ordering:

- **Kiosk Menu**: Configure kiosk menu layouts and product display
- **Kiosk Menu Group**: Organize kiosk menus into groups for different locations or times
- **QR Code Generator**: Generate QR codes for self-service ordering, allowing customers to scan and order from their mobile devices

### Future Orders

- **Future Order Close**: Manage future order closing settings. Configure when future orders can be placed and when they close for specific time periods.
