| Internal Ticket ID | Enhancement | Description
|:------------|:-----------------------------|:--------------------|
| 8697xme4k | Updated Adyen EFT interface to support Android Castles Saturn 1000 (S1F2) all in one device | Supported on Adyen specific Android S1F2 only
| 86963vdfq | Support for FreedomPay Castles Android EFT interface | Supports Castles Saturn 1000 (S1F2 & S1F4) using AFCC 1.54.0
