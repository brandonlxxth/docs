FreedomPay, Self Checkout, KMS, Clover REST Pay and Stock Transfer improvements, plus minor bug fixes.


## üöÄ Enhancements

| Internal Ticket ID | Feature | Description |
|-------------------|--------|-------------|
| 8698f5qt4 | FreedomPay EFT configuration enhancements | FreedomPay EFT interfaces now include new configuration options to control AFCC/FCC behavior. Logging during driver loading has also been improved. Users can now control more store-and-forward (offline) configuration. |
| 869a6u6yd | Clover REST Pay EFT OAuth authentication update | The interface now manages access tokens automatically to align with updated Clover OAuth security standards. |
| 869b7h07t | Honeywell Gen7 Mini DB scanner support | K2 HID drivers now support the Honeywell Gen7 Mini DB barcode scanner. |
| 869ajeazz | KMS order action buttons repositioned | KMS orders now display print and action buttons at the top of the ticket when selected. Operator experience is improved. Legacy bottom button placement remains available via configuration. |
| 8699yrcg3 | Pay at Table improvements | General fixes and improvements applied to the Pay at Table system. |
| 8699nc912 | SCO Kappture Accounts and Allowance Vouchers support | Self Checkouts now support Kappture Accounts and Allowance Vouchers, including full and partial payments. Includes an optional setting to display member account balances during transactions and settle outstanding balance |
| 86998dykz | Self-Checkout NFC and QR member identification | Self Checkouts now support scanning NFC and Kappture QR codes for member identification. |
| 8699zyeub | EFT-only Self-Checkout mode | Self Checkouts now support a mode for EFT payments only, without prompting for member identification. Not enabled by default. Member Identification Type must be set to **None**. |
| 8699yrcg3 | Operator key barcode scanning for Self-Checkout | Self Checkouts now support scanning operator key barcodes for manage order and home screen access. This speeds up the elevation process. The standard logon UI does not support barcode scanning. |
| 8699nu93f | Restricted stock transfer destinations | K-Stock Transfer now supports restricting allowed destination outlets. By default, all outlets are shown. |
