This release introduces a new "Recently Used" landing page that improves user experience by providing quick access to frequently used features. It also adds Fiscal Cloud API configuration fields for Portugal terminals and optimises user record saving performance. Bug fixes address Member token validation, Meal Deal Voucher product display with special characters, Stock Movement URL validation, Voucher availability permissions, and Terminal Yoyo discount product configuration.
