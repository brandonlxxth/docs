This release improves navigation and user experience with sidebar menu loading fixes, Kiosk Menu Group reorganization, and enhanced Kiosk Menu configuration options. It also adds performance optimizations for enterprise applications, new Self Checkout terminal options, and disables the BRISK tab to prevent accidental configuration changes. Bug fixes address Account Flash Report conversion calculations for doubles and 2-pint servings.
