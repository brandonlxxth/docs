## Overview

The Settings tab provides access to member account management, order history, security settings, and app preferences.

## Profile

Member profile displays account information:
- First name
- Last name
- Email address
- Phone number (if provided)
- Member ID

Members can update their profile information from this screen.

Profile data is stored in **Configuration > Member > Member** in the Management Portal.

## Security

Security settings allow members to manage account security:

### Change email
- Enter new email address
- Verify current password
- Confirm via verification code sent to new email

### Change password
- Enter current password
- Enter new password
- Confirm new password
- Password must meet minimum requirements

### Email notifications
Toggle email notifications for:
- Order status updates
- Promotional offers
- Account activity
- News and announcements

## Order history

View past orders with full details:

### Order list
- Order number
- Date and time
- Total amount
- Order status
- Outlet name

### Order details
Tap an order to view:
- All ordered items with modifiers
- Item prices and quantities
- Payment method used
- Collection/delivery method
- Time slot (if applicable)
- Order status history
- Receipt

### Reorder
Tap "Reorder" to add all items from a previous order to cart. Items can be modified before checkout.

Order history is stored in the Management Portal and visible under **Ordering > Dashboard** (order reports).

## Order status

View real-time status of current and recent orders:

### Current orders
Active orders show:
- Order number
- Preparation status
- Preparation point progress
- Status updates

### Status types
- **Pending** - Order received, awaiting preparation
- **In progress** - Currently being prepared
- **Ready** - Ready for collection
- **Complete** - Order fulfilled
- **Cancelled** - Order cancelled

### Push notifications
Members receive push notifications when order status changes:
- Order received
- Preparation started
- Order ready
- Delivery on the way (if applicable)

Notification settings are configured per member in the app.

## Notifications

Manage push notification preferences:

### Notification types
- **Order updates** - Status changes for orders
- **Promotional offers** - Special offers and discounts
- **News** - Board items and announcements
- **Account activity** - Balance changes, points earned

Members can enable/disable each notification type independently.

### Notification permissions
First time users are prompted to allow notifications. Permissions can be changed in device settings.

## Appearance

Customize app appearance:

### Theme
- **Light mode** - Light color scheme
- **Dark mode** - Dark color scheme
- **System** - Follow device settings

Theme preference is saved per device.

### Display options
- Font size preferences
- High contrast mode (accessibility)

## About

App information:
- App version
- Build number
- Company information
- Contact details

## Legal

Access legal documents:

### Terms and conditions
Terms of service for using the app and platform.

### Privacy policy
Privacy policy explaining data collection and usage.

### Licenses
Open source software licenses used in the app.

All legal documents are displayed in-app and can be updated by the development team.

## Configuration in Management Portal

| Settings feature | Management Portal location |
|------------------|---------------------------|
| Member profile | **Configuration > Member > Member** |
| Order history | **Ordering > Dashboard** (order reports) |
| Email notifications | Managed per member in **Configuration > Member > Member** |
| Account settings | **Configuration > Member > Account** |
