## Welcome

This is the API reference documentation for the **Kappture Integrate API v2.0.0**.

The API documentation is automatically generated from our OpenAPI specification. All endpoints are listed in the sidebar, organized by tags.

## Authentication

API endpoints require authentication using JWT tokens and API keys. Generate and manage your API keys in the [Kappture Management Portal](https://admin.dev.kappture.com) on your Profile Page.
